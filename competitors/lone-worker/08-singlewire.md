## Product lineup

- **InformaCast Fusion** — flagship mass-notification platform; hybrid cloud + on-premises architecture (cloud for management, on-prem server for IP phones / IP speakers / desktops).
- **InformaCast Mobile** — iOS + Android; receive alerts, panic-button activation, incident updates.
- **Desktop Notifier** — Windows + macOS client for full-screen takeover alerts.
- **Visitor Aware** — visitor management + threat screening add-on.
- **IP Speaker, IP Phone, Digital Signage** integrations — extends alerts into existing infrastructure.
- **Panic Buttons** — hardware (wall-mounted, USB, software hotkey) that trigger pre-built alerts; two-button models differentiate "staff assistance" from "school-wide emergency" (Alyssa-Law compliance).
- **Incident Management** — virtual response plan, group coordination, post-event reporting.

## Differentiation

Singlewire's positioning is the **Cisco IP-phone-native mass-notification leader** — InformaCast was founded by ex-Cisco engineers and remains the easiest-to-deploy mass-notification product inside an all-Cisco UC estate. Where Everbridge and AlertMedia sell multi-channel communication, Singlewire sells **in-building audio/visual takeover** of the endpoints a customer already owns.

## Strengths

- ✅ **Deep Cisco Unified Communications integration** — single-button IP-phone takeover, desk-phone push-to-talk, multi-cast paging.
- ✅ **Hybrid architecture** — cloud control plane + on-premises data plane fits regulated buyers (healthcare, K-12, government) that need on-prem call routing.
- ✅ **Single-pane-of-glass for emergency + daily communication** — overhead paging, bell schedules, daily announcements on the same platform.
- ✅ **Alyssa-Law-compliant mobile panic buttons** — two-button models differentiate staff assistance from school-wide emergency.
- ✅ **Strong customer satisfaction in K-12 + healthcare** — repeatedly a "best fit" in those vertical procurement lists.
- ✅ **No proprietary hardware required** — uses existing IP phones, speakers, digital signage.

## Weaknesses / gaps

- ❌ **No dedicated lone-worker wearable.** Mobile panic button is functional but not the core product.
- ❌ **No ARC.** Customers self-monitor.
- ❌ **Cisco dependency is a double-edged sword** — best-in-class in Cisco estates, weaker integration with Microsoft Teams / Zoom Phone estates.
- ❌ **Limited brand recognition outside North America** — minimal UK / EU presence compared to Everbridge / AlertMedia.
- ❌ **Mobile app is functional but not best-in-class** — features like location, indoor positioning, and check-ins lag CriticalArc.
- ❌ **Visitor Aware + InformaCast bundles can confuse the buyer** — they're sold together, but only ~40% of customers need visitor management.

## MyLifeLine opportunity

- **White-label mobile panic button for InformaCast estates.** Singlewire's mobile component is thin; MyLifeLine's deeper duress app can plug into InformaCast via API as a managed add-on.
- **Non-Cisco estates.** Singlewire's strength is Cisco-fluent customers; organisations standardised on Microsoft Teams or Zoom Phone need an alternative — MyLifeLine's multi-tenant SaaS plays there.
- **SMB and sub-1,000-seat deals** that Singlewire's project-led motion overlooks.
