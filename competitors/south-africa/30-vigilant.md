# Vigilant (Vigilant Care)

> **Category:** Free consumer panic app with proactive safety timers
> **Website:** https://vigilantcare.co.za

## Overview
Vigilant is a free SA personal-safety app with no app store — installed via PWA / direct install on the phone. Differentiated by **proactive safety features** (Guardian Timer / "Stay Safe" with auto-alert if no check-in, Duress PIN for covert signalling) and a **Sentry Mode** that turns the phone into a portable alarm system. WhatsApp-based family alerting.

## Core Offering
- **Panic button** → 24/7 Vigilant Control Room.
- **Guardian Timer** — set timer for high-risk moments; auto-alerts if you don't check in.
- **Duress PIN** — covert signalling if forced to cancel under duress.
- **Sentry Mode / Alarm** — phone as portable security system with motion detection + smart alarms.
- **Community Intelligence Map** — verified safety incidents near you.
- **Family features** — designated family members notified on panic.
- **WhatsApp Alerts** to family with location.

## Pricing
Free.

## Strengths vs MyLifeLine
- **Free** — pricing disruption.
- **Proactive safety timers** are genuinely useful (Uber-style "I should be home by 11" auto-alerts).
- **Duress PIN** — a covert signalling feature that addresses domestic-violence scenarios well.
- WhatsApp alerting (data-friendly) suits SA market.
- Community intelligence map is a trust-builder.

## Weaknesses vs MyLifeLine
- **App-only, PWA** — no wearable, no dedicated pendant.
- No fall detection / HR / medical-grade elderly device.
- No B2B SaaS platform.
- Free model means no revenue to scale control-room operations.
- No API / WebHooks.

## Target Customer
Budget-conscious SA users, women travelling alone, students, families wanting free safety features.
