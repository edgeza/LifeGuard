# Acticheck

- **Website**: https://acticheck.com
- **Headquarters**: UK
- **Founded**: 2014 (by entrepreneur Karl Gibbs — also founder of SureSafe)
- **Tagline**: "The Assure" — world's first continuously monitoring personal alert system wristband

## Business Model
**D2C/B2C** — direct-to-consumer, sold via yearly subscription. Unique **"I'm OK" check** model and "cold-at-home" warning feature.

## Core Products
- **Assure wristband** — continuously monitoring personal alert system
  - "Call-for-help" function
  - Fall detection
  - **"I'm OK" daily checks** (unique — user confirms wellness; otherwise escalation)
  - "Cold-at-home" warning (heating failure detection)
  - Waterproof (shower-safe)
  - **One-year battery life** (no daily charging)
- Smartphone apps (Android + iPhone) link seamlessly when away from home
- UK Response Centre as professional backup

## Pricing
- **Yearly Plan** — sold via acticheck.com/shop
- 30-day returns
- Free UK delivery
- UK-based support team

## Reach
Award-winning (Santam Security Innovation award 2018 winner — same award MyLifeLine won in 2018). UK-only D2C; founder overlap with SureSafe.

## TSA Status
Not a TSA member; smaller independent provider.

## Tech Stack
Wearable wristband + smartphone app bridge for out-of-home coverage; UK-based response centre.

## Strengths vs MyLifeLine
- **"I'm OK" daily check** is a feature no other UK competitor offers (proactive wellness vs reactive panic)
- **One-year battery life** vs MyLifeLine's 1-3 day battery — major UX win
- Award-winning (Santam Security Innovation award)
- Smart-home signal: cold-at-home warning

## Weaknesses vs MyLifeLine
- Smaller brand; no B2B SaaS / reseller network
- No 5-emergency-contact SMS workflow
- No "control room operator speaks directly to wearer" workflow (different design philosophy)
- UK-only
- Less mature supply chain vs Careline365
