# Salama Smart Safe + AURA SmartPanics (Kenya)

> **Category:** Two complementary consumer panic apps in Kenya
> **Operators:** Salama Smart Safe (Sansa Digital) · BM Security SmartPanics (BM Security)
> **Websites:** Google Play (com.olepundit.frontend) · standardmedia.co.ke/business/sci-tech/article/2001552802/

## Overview
Two distinct Kenyan consumer panic apps with similar two-tap-and-help value propositions:
- **Salama Smart Safe** (Sansa Digital, Omondi) — Kenyan-built, all-in-one personal-safety, crime-reporting, location-sharing and roadside mechanic request app. "Made for Kenya."
- **BM Security SmartPanics** (launched July 15, 2026 by BM Security) — links to BM Security's 24-hour Control Room, with panic / medical / fire / security emergency buttons, "On My Way" journey monitoring (auto-alert if user doesn't reach destination in time), "Here I Am" instant location share, multimedia incident reporting and SMS backup for poor internet.

## Core Offering
- **Salama Smart Safe** — panic alerts, crime reporting, live location sharing, emergency mechanic requests; encrypted data.
- **SmartPanics** — emergency buttons (security / medical / fire), real-time GPS to BM control room, journey monitoring, "Here I Am" location, multimedia reports, SMS fallback for poor internet, complement to BM's existing manned-guarding/alarm business.

## Pricing
Salama Smart Safe free / subscription-lite; SmartPanics pricing via BM Security service bundles.

## Strengths vs MyLifeLine
- **Kenyan-first localisation** and BM Security's established control-room infrastructure.
- Journey monitoring ("On My Way") is genuine differentiator matching Namola's "Places" feature but Kenya-tuned.
- SMS fallback is real — most competitors die on poor connectivity.
- Comprehensive incident capture (photo / video / audio).

## Weaknesses vs MyLifeLine
- **App-only** — no medical pendant, no fall detection.
- No medical/allergy operator profile.
- Kenya-only.
- SmartPanics barely launched; production scale yet to be proven.

## Target Customer
Kenyan families, commuters, BM Security alarm / guard customers, estate residents.
